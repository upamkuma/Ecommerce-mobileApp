# React Native E-Commerce Mobile App

A simple **E-Commerce Mobile Application** built using **React Native and Expo Router**.
This app allows users to browse products, add items to a cart, save products to a wishlist, and view a basic profile page.

---

## Features

* 🛍 Product Listing
* 🛒 Add to Cart functionality
* ❤️ Wishlist functionality
* 👤 User Profile section
* 📱 Tab-based navigation using Expo Router
* ⚡ State management using React Context API

---

## Technologies Used

* React Native
* Expo
* Expo Router
* JavaScript / TypeScript
* React Context API

---

## Project Structure

```
app
 ├ (tabs)
 │   ├ index.tsx        # Products page
 │   ├ cart.tsx         # Cart page
 │   ├ wishlist.tsx     # Wishlist page
 │   └ profile.tsx      # Profile page

components
 └ ProductCard.js       # Product UI component

context
 └ StoreContext.js      # Global state for cart and wishlist

assets                  # Images and static files
```

## Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/upamkuma/Ecommerce-mobileApp.git
```

### 2️⃣ Go to the project folder

```
cd Ecommerce-mobileApp
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Run the application

```
npm start
```
